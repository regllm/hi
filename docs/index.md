# LLM

A command-line utility for interacting with Large Language Models, such as OpenAI's GPT series.

## Quick start

```bash
# Install LLM
pip install llm
# Or use: brew install simonw/llm/llm

# Paste your OpenAI API key into this:
llm keys set openai

# Run a prompt
llm "Ten fun names for a pet pelican"
```
